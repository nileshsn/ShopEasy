# Email Templates

This directory contains email templates for ShopEasy.

## Confirmation Email Template

### Files:
- `confirmation.html` - HTML version for rich email clients
- `confirmation.txt` - Plain text version for fallback

### Usage in Supabase:

1. **Go to Supabase Dashboard** ‚Üí Your Project
2. **Navigate to:** Authentication ‚Üí Email Templates
3. **Select:** Confirm signup
4. **Replace the content** with the HTML from `confirmation.html`

### Template Variables:

The following variables are available in Supabase email templates:

- `{{ .ConfirmationURL }}` - The confirmation link (generated by Supabase)
- `{{ .Email }}` - User's email address
- `{{ .Token }}` - Confirmation token
- `{{ .SiteURL }}` - Your site URL

### Best Practices:

1. Always include both HTML and text versions
2. Use clear, professional language
3. Make the CTA button prominent
4. Include security notes to build trust
5. Test email rendering across clients (Gmail, Outlook, Apple Mail, etc.)
6. Keep file size under 100KB
7. Use inline CSS for better email client compatibility
